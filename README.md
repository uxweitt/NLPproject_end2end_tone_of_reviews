# End2end nlp проект по обработке тональности отзывов с помощью RNN

## Данный проект представляет собой end-to-end pet-проект для определения тональности отзывов.
___

Датасет был взят с [kaggle](https://www.kaggle.com/datasets/mikhailklemin/kinopoisks-movies-reviews/data). Он имеет структуру:
```
dataset/
├── neg/
│   ├── 1000083-0.txt
│   ├── 1000083-1.txt
│   └── ...
├── neu/
│   ├── 1000023-3.txt
│   ├── 1000125-0.txt
│   └── ...
└── pos/
    ├── 1000023-0.txt
    ├── 1000023-1.txt
    └── ...
```
