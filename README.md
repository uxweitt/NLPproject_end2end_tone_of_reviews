# End2end nlp проект по обработке тональности отзывов с помощью RNN(GRU)

## Resume

Проект был реализован с целью изучить инструмент (PyTorch) и понять логику работы RNN (GRU), в данной работе была реализация собственного датасета (torch.utils.data.Dataset), класса препроцессинга, цикла обучения инференса модели, и написание примитивного веб приложения на FastApi с возможностью запустить проект в контейнере.
___

Датасет был взят с [kaggle](https://www.kaggle.com/datasets/mikhailklemin/kinopoisks-movies-reviews/data). Он имеет следующую структуру:
```
dataset/
├── neg/
│   ├── 1000083-0.txt
│   ├── 1000083-1.txt
│   └── ...
├── neu/
│   ├── 1000023-3.txt
│   ├── 1000125-0.txt
│   └── ...
└── pos/
    ├── 1000023-0.txt
    ├── 1000023-1.txt
    └── ...
```
_________

**Модель** - двухслойная сеть. Первый слой - двунаправленная GRU, второй слой полносвязный.
________
Установка зависимостей:
`uv pip install -r requirements.txt`
*На моей рабочей станции использовался **torch2.11.0+cu126**
____
Поднять докер можно с помощью команды:
`docker-compose up --build`
